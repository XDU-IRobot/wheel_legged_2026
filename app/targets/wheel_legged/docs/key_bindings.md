# wheel_legged 键位绑定文档

本文档列出 wheel_legged 目标下所有输入设备的键位映射。

输入优先级：**VT03 图传键鼠（CAN 桥）> DR16 遥控器**

---

## 1. DR16 遥控器

### 1.1 拨杆

| 输入 | 位置 | 功能 |
|------|------|------|
| 左拨杆 (switch_l) | 上 | 战斗域 (kCombat) |
| | 中 | 维护域 (kService) |
| | 下 | 关闭 (kDisabled)；配合右拨杆进入辨识/验证模式 |
| 右拨杆 (switch_r) | 上 | 高腿长 (kHigh) / 战斗域下：kAutoAimFuSmall |
| | 中 | 中腿长 (kMid) / 战斗域下：kAutoAimAmmo |
| | 下 | 低腿长 (kLow) / 战斗域下：kNormal |

> **左拨杆下 + 右拨杆组合**（仅 DR16 兜底路径）：
> - 右上 → Service 域 + 云台辨识模式 (kIdent)
> - 右中 → Service 域 + 前馈验证模式 (kFfVerify)
> - 右下 → kDisabled

### 1.2 摇杆

| 输入 | 功能 |
|------|------|
| 右摇杆 Y | 前进/后退（归一化 [-1, 1]） |
| 右摇杆 X | 左右平移（归一化 [-1, 1]） |
| 左摇杆 X | 云台偏航速率积分 |
| 左摇杆 Y | 云台俯仰速率积分 |

### 1.3 鼠标

| 输入 | 功能 |
|------|------|
| 鼠标 X | 云台偏航速率积分 |
| 鼠标 Y | 云台俯仰速率积分 |
| 鼠标左键 | （未分配） |
| 鼠标右键 | 按住进入自瞄模式（电平有效） |

### 1.4 拨轮 (dial)

| 输入 | 功能 |
|------|------|
| 拨轮 >= 阈值 | 小陀螺保持 |
| 拨轮从中心拨至负阈值 | 跳跃触发（战斗域下屏蔽） |

## 2. VT03 图传键鼠（CAN 桥转发）

### 2.1 移动

| 键位 | 功能 |
|------|------|
| W | 前进 |
| S | 后退 |
| A | 左平移 |
| D | 右平移 |

### 2.2 工作域与腿长

| 键位 | 触发方式 | 功能 |
|------|----------|------|
| Q | 上升沿 | 工作域循环：kDisabled → kCombat → kDisabled … |
| C | 上升沿 | 切换中腿长；已在中腿长则回低腿长 |
| V | 上升沿 | 切换高腿长（1 次上台阶）；已在高腿长则回低腿长 |
| B | 上升沿 | 切换高腿长（2 次上台阶模式）；已在高腿长则回低腿长 |

### 2.3 运动控制

| 键位 | 触发方式 | 功能 |
|------|----------|------|
| Shift | 电平 | 小陀螺保持（按住生效） |
| F | 上升沿 | 跳跃（Ctrl 按住时屏蔽） |
| R | 上升沿 | 重置底盘正方向（偏航归零） |

### 2.4 摩擦轮调速

| 键位 | 触发方式 | 功能 |
|------|----------|------|
| Ctrl+Z | 上升沿 | 摩擦轮目标转速 -20 rpm |
| Ctrl+X | 上升沿 | 摩擦轮目标转速 +20 rpm |

### 2.5 自瞄模式切换

| 键位 | 触发方式 | 功能 |
|------|----------|------|
| 鼠标右键 | 电平 | 按住进入自瞄模式（子模式由 aim_mode 决定） |
| Ctrl+F | 上升沿 | 切换右键自瞄子模式：kAmmo ↔ kFuSmall（从 kFuBig 回 kAmmo） |
| Ctrl+G | 上升沿 | 切换右键自瞄子模式：kAmmo ↔ kFuBig（从 kFuSmall 回 kAmmo） |

**aim_mode 状态机**：

```
         Ctrl+F                Ctrl+G
kAmmo ──────────► kFuSmall    kAmmo ──────────► kFuBig
  ▲                  │          ▲                  │
  │    Ctrl+F        │          │    Ctrl+G        │
  └──────────────────┘          └──────────────────┘

从 kFuBig 按 Ctrl+F → kAmmo
从 kFuSmall 按 Ctrl+G → kAmmo
```

| aim_mode | 右键按住时 combat_profile | aimbot_mode 发送值 |
|----------|---------------------------|-------------------|
| kAmmo | kAutoAimAmmo | 1（打装甲板） |
| kFuSmall | kAutoAimFuSmall | 2（打小符） |
| kFuBig | kAutoAimFuBig | 3（打大符） |

### 2.6 其他

| 键位 | 触发方式 | 功能 |
|------|----------|------|
| E | 电平 | UI 刷新使能（按住时触发 UI 重绘） |
| Z | 长按 1s | 切换倒地自启自动/手动模式 |
| G | 长按 1s | 切换 DR16 并行模式（Ctrl 按住时屏蔽） |

### 2.7 鼠标（云台控制）

| 输入 | 功能 |
|------|------|
| 鼠标 X | 云台偏航速率积分 |
| 鼠标 Y | 云台俯仰速率积分 |
| 鼠标左键 | （未分配） |
| 鼠标右键 | 按住进入自瞄模式 |

## 3. DR16 并行模式

开启后（G 键长按 1s），当 VT03 键盘无输入时，DR16 的拨杆/拨轮/摇杆可叠加生效：

| DR16 输入 | 并行模式下的行为 |
|-----------|-----------------|
| 左拨杆 | 覆盖工作域（仅当键盘域为 kDisabled 时） |
| 右拨杆 | 覆盖腿长（仅当键盘腿长为 kLow 时） |
| 拨轮上拨 | 小陀螺（叠加） |
| 拨轮下拨 | 跳跃（叠加，战斗域下屏蔽） |
| 右摇杆 | 移动（叠加，仅当键盘方向无输入时） |
| 左摇杆 | 云台积分（叠加，仅当鼠标静止时） |
