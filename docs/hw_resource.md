# 硬件资源分配

## GPIO

| 资源   | 引脚模式    | 说明               |
|------|---------|------------------|
| PA15 | 上升沿外部中断 | 板载的按键            |
| PC13 | 推挽输出    | 驱动板载的可控24V电源 #1  |
| PC14 | 推挽输出    | 驱动板载的可控24V电源 #2  |
| PC15 | 推挽输出    | 驱动板载的可控5V电源      |
| PC0  | 推挽输出    | BMI088 加速度计的片选信号 |
| PC3  | 推挽输出    | BMI088 陀螺仪的片选信号  |

## TIM

| 资源    | 说明                    |
|-------|-----------------------|
| TIM12 | 驱动板载蜂鸣器               |
| TIM13 | 驱动程序主循环，配置为定频2khz触发中断 |

## FDCAN

| 资源     | 说明                                 |
|--------|------------------------------------|
| FDCAN1 | 用于一般用途，配置为经典CAN模式，波特率1M            |
| FDCAN2 | 同上                                 |
| FDCAN3 | 用于和上位机通信，配置为FD模式，仲裁波特率1M，数据波特率3.2M |

## SPI

| 资源   | 说明                        |
|------|---------------------------|
| SPI1 | 空闲，配置到板载的10pin接口，可以用于外接设备 |
| SPI2 | 驱动板载的BMI088 IMU           |
| SPI6 | 驱动板载的WS2812彩灯             |

## UART

| 资源          | 参数          | 说明       |
|-------------|-------------|----------|
| UART5       | 100000, 9E2 | SBUS，遥控器 |
| UART2-RS485 | -           | 空闲       |
| UART3-RS485 | -           | 空闲       |
| UART1       | -           | 空闲       |
| UART7       | -           | 空闲       |
| UART10      | -           | 空闲       |

## ADC

| 资源   | 说明                          |
|------|-----------------------------|
| ADC1 | IN4 配置到板载的XT30分压电路，用于监测电池电压 |
